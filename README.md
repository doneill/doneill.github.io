# jdONeill
![Build](https://github.com/doneill/doneill.github.io/actions/workflows/build.yml/badge.svg)

GitHub pages site based on [Hexo](https://hexo.io/) for [jdONeill](https://jdoneill.com)

```bash
# install hexo-cli
$ npm install hexo-cli -g

# clean db
$ hexo clean

# generate site
$ hexo generate

# start server
$ hexo server
```
