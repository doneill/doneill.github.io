Helix JVM language setup

jdONeill
2025-01-06
helix, jvm, kotlin, scala

I have been a huge helix fan for some time now. It is my daily coding driver for most languages and environments. Helix support many languages and language servers out of the box and are easy to configure. In this post I’ll step through basic configuration to work with Java and Kotlin in Helix.

Java Development

Install Prerequisites

  • JDK (Java Developer Kit) on your system

  • Install jdtls (Eclipse JDT Language Server) for Java language support

  • Configure Helix language support, add to ~/.config/helix/languages.toml

Kotlin Development

Install Prerequisites

  • Install Kotlin
  • Install Kotlin Language Server
  • Configure Helix language support, add to ~/.config/helix/languages.toml

BONUS Scala Development

Install Prerequisites

  • Configure Helix language support, add to ~/.config/helix/languages.toml
